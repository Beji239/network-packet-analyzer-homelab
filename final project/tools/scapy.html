Scapy

Python Packet Manipulation Library

Scapy is a powerful interactive packet manipulation program written in Python. It can forge or decode packets of a wide number of protocols, send them on the wire, capture them, match requests and replies, and much more. It’s the go-to library for network research, penetration testing, and building custom security tools.

Core Use Cases

- Packet Crafting: Build custom packets layer by layer for testing firewalls/IDS
- Packet Sniffing: Capture live traffic with programmable filters
- Protocol Dissection: Decode Ethernet, IP, TCP, UDP, DNS, HTTP, etc.
- Network Scanning: Write custom scanners beyond what Nmap offers
- Attack Simulation: Generate SYN floods, ARP spoofing, DNS poisoning for lab testing
- PCAP Analysis: Read/write .pcap files and extract forensic evidence

Code Examples

1. Basic packet sniffer:

from scapy.all import sniff, IP

def packet_callback(pkt):
  if IP in pkt:
    print(f"{pkt[IP].src} -> {pkt[IP].dst}")

sniff(prn=packet_callback, count=10)

Captures 10 packets and prints source/destination IPs. Foundation of any sniffer.

2. Craft and send a TCP SYN packet:

from scapy.all import IP, TCP, sr1

pkt = IP(dst="192.168.1.1")/TCP(dport=80, flags="S")
response = sr1(pkt, timeout=2)
response.show()

Manually builds a SYN packet to test if port 80 is open. Useful for stealth scans and firewall testing.

3. Detect SYN flood by counting SYNs per source IP:

from scapy.all import sniff, TCP
from collections import Counter

syn_counter = Counter()

def detect_syn_flood(pkt):
  if TCP in pkt and pkt[TCP].flags == "S":
    syn_counter[pkt[IP].src] += 1
    if syn_counter[pkt[IP].src] > 100:
      print(f"[!] Possible SYN flood from {pkt[IP].src}")

sniff(filter="tcp", prn=detect_syn_flood)

Alerts when a single IP sends >100 SYN packets. Logic I used in my packet sniffer for anomaly detection.

Defensive Applications

Blue teams use Scapy to parse logs, analyze malware C2 traffic from PCAPs, and build custom detection scripts. You can replay suspicious traffic in a sandbox, fuzz-test IDS rules, or generate baseline traffic to tune SIEM alerts. It’s faster than GUI tools for automation.
