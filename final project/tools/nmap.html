Nmap

Network Mapper & Port Scanner

Nmap is a free, open-source utility for network discovery and security auditing. Security teams use it to discover hosts, identify open ports, detect running services, fingerprint operating systems, and find vulnerabilities. It's the first tool used in both penetration tests and defensive network inventory.

Core Use Cases

- Host Discovery: Find live devices on a network without scanning every port
- Port Scanning: Identify open TCP/UDP ports and services
- Service Detection: Determine application name and version running on ports
- OS Fingerprinting: Detect operating system and hardware of targets
- Vulnerability Scanning: Run NSE scripts to check for known CVEs
- Firewall Evasion: Test IDS/IPS rules and firewall configurations

Common Scan Examples

1. Basic stealth SYN scan:

nmap -sS 192.168.1.0/24

Scans entire subnet for live hosts using half-open SYN packets. Fast and less likely to be logged.

2. Service/version detection:

nmap -sV -p 1-1000 10.0.0.15

Probes ports 1-1000 and identifies service versions like Apache 2.4.41 or OpenSSH 8.2p1.

3. Aggressive scan with OS detection + scripts:

nmap -A -T4 scanme.nmap.org

-A enables OS detection, version detection, script scanning, and traceroute. -T4 for faster execution.

4. Check for vulnerable SMB version:

nmap --script smb-vuln-ms17-010 -p 445 192.168.1.10

Uses NSE to check if target is vulnerable to EternalBlue. Critical for patch management.

Defensive Applications

Blue teams run scheduled Nmap scans to maintain asset inventory, verify that only approved services are exposed, and detect rogue devices. Comparing scan results over time reveals unauthorized changes. Rate-limiting and IDS tuning prevent scan detection from causing alert fatigue.

My Experience

I use Nmap in my home lab to map my virtual network before and after deploying new VMs. This verifies firewall rules in pfSense are working and no unintended ports are exposed. I also practice parsing Nmap XML output with Python to auto-generate network diagrams and feed results into my packet sniffer for targeted analysis.
