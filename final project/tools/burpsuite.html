Burp Suite

Web Application Security Testing Platform

Burp Suite is the industry-standard toolkit for testing web application security. It acts as an intercepting proxy, letting you inspect and modify traffic between your browser and target web servers. Both red teams and blue teams use it: pentesters find vulnerabilities, defenders reproduce attacks and validate WAF rules. Community Edition is free; Pro adds automated scanning.

Core Modules

- Proxy: Intercept HTTP/S requests and responses. Modify headers, cookies, parameters in real-time
- Repeater: Manually resend requests with tweaked payloads to test for SQLi, XSS, auth bypass
- Intruder: Automate fuzzing attacks with custom payload lists for brute force and enumeration
- Scanner: [Pro] Automated crawl + audit for 100+ vulnerability types
- Decoder: Encode/decode Base64, URL, HTML, JWT tokens
- Comparer: Diff responses to find subtle logic flaws

Common Workflows

1. Intercept login request to test for SQL injection:

POST /login HTTP/1.1
Host: target.com
Content-Type: application/x-www-form-urlencoded

username=admin' OR '1'='1--&password=test

Use Repeater to send modified payloads. Check response for auth bypass or error-based SQLi.

2. Brute force directories with Intruder:

GET /§admin§ HTTP/1.1
Host: target.com

Payload list: admin, dashboard, config,.git, backup.zip
