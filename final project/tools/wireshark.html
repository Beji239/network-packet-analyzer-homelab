Wireshark

Network Protocol Analyzer

Wireshark is the world's most widely-used network protocol analyzer. It lets you see what's happening on your network at a microscopic level and is the de facto standard across many industries and educational institutions.

Use Cases

- Capture live packet data from network interfaces
- Display packets with detailed protocol information
- Filter traffic to isolate suspicious activity
- Follow TCP streams to reconstruct sessions
- Detect malware C2 beacons and data exfiltration

Example: Filter for SYN Floods

tcp.flags.syn == 1 and tcp.flags.ack == 0

My Experience

Used Wireshark extensively while building my packet sniffer to validate that my Scapy code was parsing headers correctly. Analyzed PCAPs from my home lab to identify DNS tunneling attempts and unusual outbound connections.
