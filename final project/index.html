Case File 001: Packet Sniffer

ACTIVE

Built a network packet sniffer to capture and analyze live traffic. Uses Npcap for Windows packet capture and Scapy for protocol dissection. Generates real-time graphs showing protocol distribution, top talkers, and anomalous traffic patterns. Detects SYN floods and port scans by analyzing packet frequency.

from scapy.all import sniff, IP, TCP
import matplotlib.pyplot as plt

def packet_handler(pkt):

sniff(prn=packet_handler, store=0)
